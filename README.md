# XamlBrewer.WinUI3.QuestPDF.Sample
Demonstrates generating PDF documents with [QuestPDF](https://github.com/QuestPDF/QuestPDF) in a WinUI 3 Desktop application:
* Hello World
* Invoice with tables
* Chart
* Barcode
* (more to come)
